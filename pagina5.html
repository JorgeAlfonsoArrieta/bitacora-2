REFERENCIAS AVANZADAS

-Atributo id: Se utiliza para identificar un elemento, este debe ser único dentro de la página.

-Atributo class:Este atributo es más flexible y puede ser asignado a varios elementos para compartir un diseño similar.

-Elementos: Se refiere a la selección de elementos HTML basados en sus nombres de etiqueta.

-Pseudoclases:Las pseudoclases son utilizadas para definir un estado especial de un elemento.

-Selector de atributo:Permite seleccionar elementos en función de la presencia o valor de sus atributos.


RESUMEN

En CSS3, el atributo id se utiliza para identificar de manera única un elemento en el HTML, mientras que el atributo class permite agrupar múltiples elementos. Se pueden seleccionar elementos por su nombre de etiqueta, y las pseudoclases como :nth-child() y :first-child permiten aplicar estilos a elementos según su posición dentro de su padre. Además, los selectores de atributos permiten estilizar elementos basándose en la presencia o el valor de sus atributos, lo que proporciona flexibilidad y control sobre la apariencia de los elementos en la página.
