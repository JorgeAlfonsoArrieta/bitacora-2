IDEAS

-Animation-name: Ayuda a asignarle un nombre a la animacion.

-Animation-duration: Especifica el tiempo para completar un ciclo.

-Animation-iteration-count: Da el numero de veces que se tiene que repetir la animación.

-Animation-delay: Da un retraso para el inicio de una animación.

-Animation-play-state:Especifica si la animación se esta ejecutando o está en pausa.

ANIMACIÓN EN HTML

La animación en html utiliza @keyframes. Esta animación cambia la posición del objeto seleccionado. Para que una animación funcione se le debe vincular la animación a un elemento.

También es importante especificar la propiedad de duración de la animación, si no se hace no se producirá ninguna animación.


RESUMEN

Las imágenes en HTML se utilizan para mostrar contenido visual en las páginas web, como fotos y gráficos, mediante el elemento < img>, que no requiere etiqueta de cierre. El atributo src especifica la ruta o URL de la imagen, mientras que el atributo alt proporciona un texto alternativo que es esencial para la accesibilidad.
