IDEAS

-Border-radius corner: Esta propiedad genera esquinas redondeadas para la caja formada por el elemento. Posee dos parámetros diferentes que dan forma a la esquina.

-Border-shadow: Esta propiedad crea sombraspara la caja formada por el elemento.

-@Font-face: Esta regla nos permite cargar y usar cualquier fuente que necesitemos.

-Linear-gradient: Esta función puede ser aplicada a las propiedades background para generar un gradiente lineal.

-RGBA: Esta función es una mejora del rgb con un canal alfa que especifica la opacidad de un color.

-HSLA: Significa tono, saturación y luminosidad más un canal alfa que especifica la opacidad para un color.

-Outline: Genera un segundo borde alejado del elemento.

-Transform: Esta función modifica la forma de un elemento.

NUEVAS PROPIEDADES

Es un intento de reducir el código Javascript y para estandarizar funciones populares, CSS3 no solo cubre diseño y estilos web , sino también forma y movimiento

Desde esquinas redondeadas y sombras hasta transformaciones y reposicionamientos de los elementos ya presentadosen pantalla, cada posible efecto aplicado previamente utilizando Javascript fue cubierto.


RESUMEN

CSS3 trajo consigo un mundo de posibilidades creativas para los diseñadores web. Con los gradientes, ahora podemos crear fondos vibrantes y fluidos sin necesidad de recurrir a imágenes, usando linear-gradient y radial-gradient para hacer transiciones de color que dan vida a cualquier diseño. También tenemos las sombras, tanto para texto como para elementos, que añaden una sensación de profundidad y elegancia a nuestras páginas. Las transiciones y transformaciones permiten que los elementos cobren vida al interactuar con ellos, creando animaciones suaves y atractivas. Y no podemos olvidar la magia de @font-face, que nos permite incorporar fuentes personalizadas, haciendo que la tipografía sea tan única y estilizada como deseamos. Todo esto convierte la experiencia de diseño en algo mucho más emocionante y visualmente atractivo.
